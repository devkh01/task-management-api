# Task Management API

A simple REST API built using Node.js and Express.js.

## Features

- Create a task
- Get all tasks
- Get a task by ID
- Update a task
- Delete a task

## Tech Stack

- Node.js
- Express.js

## Installation

```bash
npm install
```

## Run

```bash
npm run dev
```

## API Endpoints

| Method | Endpoint |
|---------|----------|
| GET | /tasks |
| GET | /tasks/:id |
| POST | /tasks |
| PUT | /tasks/:id |
| DELETE | /tasks/:id |
